# 📝 NoteFlow

A mini notes application inspired by Google Keep built using PHP, JavaScript, AJAX, and MySQL.

## ✨ Features
- Create, edit, and delete notes
- Live search functionality
- Auto-save using AJAX
- Responsive and clean UI
- MySQL database integration

## 🛠️ Tech Stack
- PHP
- JavaScript
- AJAX (Fetch API)
- MySQL
- HTML & CSS

## 📌 Purpose
This project was created to practice CRUD operations, database management, AJAX requests, and improving user experience with real-time interactions.

## 🚀 Future Improvements
- User authentication
- Dark mode
- Note categories/tags
- Pin favorite notes
- Rich text editor
